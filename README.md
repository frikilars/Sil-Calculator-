SIL CALCULATOR 

Sil Calculator adalah aplikasi kalkulator web modern yang responsif, dibangun menggunakan JavaScript murni dan Tailwind CSS. Aplikasi ini menyediakan fitur memori lengkap, panel histori operasi, dukungan keyboard penuh, serta sistem perhitungan dengan prioritas operator matematis yang akurat.

Keunggulan Aplikasi
1. Kalkulasi Matematis Lengkap
Mendukung operasi dasar:

Penjumlahan (+)

Pengurangan (−)

Perkalian (×)

Pembagian (÷)

3. Sistem Perhitungan Cerdas (PEMDAS/BODMAS)
Aplikasi sudah menghitung dengan urutan prioritas yang benar.
Contoh:
8 + 4 × 3 = 20
(4 × 3 = 12, lalu 8 + 12 = 20)

4. Fitur Penyimpanan Memori
Aplikasi memiliki 4 tombol memori:

M+ : Menambah nilai layar ke memori

M− : Mengurangi nilai layar dari memori

MR : Memanggil nilai memori ke layar

MC : Menghapus memori

4. Panel Histori
Berada di sisi kana layar (desktop)

Menyimpan 5 operasi terbaru

Menampilkan ekspresi lengkap dan hasilnya

Dilengkapi tombol Hapus Semua Histori

6. Kontrol via Keyboard: Semua tombol dapat diakses menggunakan keyboard
   
7. Proteksi Error
Tidak bisa memasukkan desimal ganda

Handling pembagian nol

Notifikasi error saat ekspresi tidak valid

CE, C, dan backspace berfungsi normal

9. Tampilan Modern & Responsif
Menggunakan Tailwind CSS

Layout adaptif (mobile — tablet — desktop)

Histori otomatis bergeser ke atas pada ukuran layar kecil

Shadow lembut dan font Inter
